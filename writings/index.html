Cracked: Github keeps asking for username and password in terminal

13 October 2014

Turns out, this was an outcome of a mistake I made during initializing my Github repository. When initializing github’s remote (origin), make sure to use SSH clone URL instead of using the normal URL. You can find your SSH URl by simply visiting your repository’s github page. It will be of the form git@github.com:username/repository-name.git.

Cracked: how to bundle and run terminal commands

13 October 2014

I needed to figure out a clever to way to deploy my octopress blog posts by running just one command.

Here’s how I did it:

1) Open your favorite text editor and enter your terminal commands (one per line). Save it with a .sh extension. Let’s call it file.sh.

2) Open your terminal and enter sudo chmod 700 file.sh

3) In order to run the command bundle enter ./file.sh
