Mean.IO learnings

25 February 2015

“Any application that can be written in Javascript will be eventually written in Javascript” - [Atwoods Law] (http://blog.codinghorror.com/the-principle-of-least-power/)

Without a doubt, JS is becoming the dominant language of the web. It didn’t require much thinking to decide that Garuda Robotics will utilize a javascript based stack. After some searching, we decided to go with Linnovate’s MEAN.IO. Linnovate is a web development services business in Israel and they are the guys behing MEAN.IO. Since inception, they’re getting 100s of downloads a day and have an increasingly growing community. They’ve become so popular now that Google Cloud has a Click to Deploy feature for MEAN.IO apps.

MEAN.IO CORE - Mongo, Express, Node, Angular JS CORE MODULES - DB, MENU, CONFIG… CORE PACKAGES - Articles, Users, System, CLI, Themes

In MEAN.IO, everything is a package. MEAN.IO provides some core packages important for the application. You can create your custom packages as well relevant to your product.
